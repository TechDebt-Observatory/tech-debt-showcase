# Tech Debt Observatory

Systematic documentation framework for legacy C/C++ codebases across the Linux/BSD open source ecosystem.

## Mission

Transform industry practices around code documentation by making legacy code understandable to both humans and AI systems, addressing the fundamental problem that undocumented code creates barriers for new maintainer onboarding and AI-assisted vulnerability detection.

## Repository Structure

- **`framework/`** - 6-dimensional documentation specification
- **`methodology/`** - Research methodology and protocols
- **`tools/`** - Automation scripts for applying the framework
- **`showcase/`** - Static HTML showcase site
- **`case-studies/`** - Full CVE analyses (private, git-ignored)
- **`research/`** - Research data and drafts (private, git-ignored)
- **`local-repos/`** - Reference repositories (private, git-ignored)
