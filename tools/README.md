# Analysis and Documentation Tools

Scripts for applying the 6D framework and analyzing codebases.

## Scripts

- `analyze-comments.sh` - Analyze comment coverage in CVE-affected files
- `prepare-worst-cve.sh` - Prepare files for documentation
- `setup-pipeline.sh` - Set up the analysis pipeline
- `quickstart.sh` - Quick start guide

## Pipeline

Python scripts for processing and rendering analysis results.
